import { Tabs, TabItem } from '@astrojs/starlight/components';

You can install library via package manager.

<Tabs>
  <TabItem label='npm'>
```bash
npm install --save react-google-map-wrapper
```
  </TabItem>
  <TabItem label='yarn'>
```bash
yarn add react-google-map-wrapper
```
  </TabItem>
  <TabItem label='pnpm'>
```bash
pnpm install react-google-map-wrapper
```
  </TabItem>
</Tabs>

> **If you're using React v18, please install version `1.0.7`.**


> If you want to see a example before download? Try out the [basic example](/react-google-map-wrapper/docs/examples/basic).

### TypeScript

In TypeScript, we recommend installing `@types/google.maps`

<Tabs>
  <TabItem label='npm'>
```bash
npm install -D @types/google.maps
```
  </TabItem>
  <TabItem label='yarn'>
```bash
yarn add -D react-google-map-wrapper
```
  </TabItem>
  <TabItem label='pnpm'>
```bash
pnpm install -D react-google-map-wrapper
```
  </TabItem>
</Tabs>
